Container

화면의 폭에 따라 내용의 최대 폭과 최소 폭을 제한할 수 있는 컴포넌트입니다.

Container 컴포넌트입니다!

주로 웹 사이트의 주요 컨텐츠의 표시를 한정하는데에 사용합니다.


import React from "react";
import { Container } from "@solo5star/layout-component";

const App = () => {
  return (
    <Container minWidth={320} maxWidth={480} style={{ background: "yellow" }}>
      <div style={{ padding: '20px' }}>
        <h2>Container 컴포넌트입니다!</h2>
        <p>주로 웹 사이트의 주요 컨텐츠의 표시를 한정하는데에 사용합니다.</p>
      </div>
    </Container>
  );
};

export default App;
  • maxWidth 와 minWidth를 설정하여 컨텐츠가 표시될 최소 혹은 최대 폭을 조절할 수 있습니다.

maxWidth 는 사용자가 몰입할 수 있을 정도로 컨텐츠를 모아줄 수 있는 값을 설정하시면 좋습니다.
