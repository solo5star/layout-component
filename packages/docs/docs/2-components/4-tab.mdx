# Tab

여러 개의 자식 컴포넌트를 전환하면서 볼 수 있도록 해주는 컴포넌트입니다.

---

  <Tab />
</Example>

<br/>

<CodeBlock language="tsx">
  {TabSource}
</CodeBlock>

* `tabs` 속성으로 탭 버튼들을 만들 수 있습니다. 탭 버튼과 자식 컴포넌트는 순차적으로 매칭됩니다.

`<Tab />` 의 자식으로 Fragment(`<></>`)를 넣으면 유의미한 자식 요소가 나올 때 까지 재귀로 탐색합니다.

예를 들어, 다음의 두 코드에선 `<div />` 를 3개로 정상적으로 인식합니다.

```tsx
<>
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</>
```

```tsx
<>
  <>
    <div>Item 1</div>
    <div>Item 2</div>
    <div>Item 3</div>
  </>
</>
```
